# ??? 我写了一个命令行工具，让你的日常工作\"顺手\"起来\n\n\u003e 11个文件，874行代码，5个实用模块——一个轻量但好用的 Python CLI 工具箱。\n\n---\n\n## 为什么要写这个？\n\n日常开发中总有一些重复性操作让人头疼：\n\n- 老板丢来 50 张图，要你统一调尺寸、加水印\n- 下载文件夹乱成一团，找文件全靠搜索\n- 想把 Markdown 笔记转成图片发朋友圈，还得截图\n- 从网页里扒链接，一个个复制粘贴\n\n这些小事不值得打开 PS，也不值得写个临时脚本——但就是烦。\n\n于是有了 **[顺手 (ShunShou)](https://github.com/wangbeiebi/shunshou)**。\n\n---\n\n## 它能干什么？\n\n```bash\n# 一键批量调整图片尺寸\nss image resize ./photos --width 800 --format webp --quality 80\n\n# 一键加水印\nss image watermark ./photos --text \"?2024\" --position bottom-right\n\n# 一键整理文件夹（按类型/日期自动归类）\nss organize ~/Downloads\n\n# Markdown 直接渲染成图片！\nss md2img README.md -o output.png --theme dark\n\n# 从网页批量提取所有链接\nss links https://example.com\n\n# 剪贴板去重\nss clip dedup\n```\n\n---\n\n## 技术选型\n\n| 依赖 | 用途 |\n|------|------|\n| **Click** | 命令行框架，嵌套子命令 + 进度条 |\n| **Pillow** | 图片处理，包括水印、渲染 MD 为图 |\n| **BeautifulSoup** | 网页链接提取 |\n| **标准库** | pathlib, shutil, subprocess... |\n\n故意做得轻——不需要 Node、不需要 Docker、不需要 GPU。\n装一个 `pip install shunshou` 就能用。\n\n---\n\n## 核心亮点\n\n### ??? 图片批处理\n支持 resize / watermark / convert 三种操作，自动识别目录下所有图片，进度条显示处理状态。水印功能自动加载系统自带中文字体（微软雅黑/PingFang/Noto Sans CJK 全覆盖）。\n\n```python\n# 核心就这么几行\nimg = Image.open(path).convert(\"RGBA\")\noverlay = Image.new(\"RGBA\", img.size, (0, 0, 0, 0))\ndraw = ImageDraw.Draw(overlay)\ndraw.text((x, y), \"?水印\", font=font, fill=(255, 255, 255, alpha))\nresult = Image.alpha_composite(img, overlay)\n```\n\n### ?? 智能文件整理\n按扩展名自动归类为 Images / Documents / Archives / Videos / Audio / Code / Other 七大类。还支持按修改日期按月归档。支持 `--dry-run` 预览模式，不怕误操作。\n\n### ??
